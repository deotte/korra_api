Legend of Korra API

An open source REST API inspired by the Legend of Korra

Check out the project on Github

Available Routes

Route

GET /api/characters/:id

Description

Returns a specific character by ID

Example Link

/api/characters/1

Example Response

							
HTTP 200 OK
Content-Type: application/json
{
	"type":"Human",
	"gender":"Woman",
	"image_url":"https://static.wikia.nocookie.net/avatar/images/3/31/Korra_smiling.png/revision/latest?cb=20200907192928",
	"first_episode":"Welcome to Republic City",
	"nationality":["Wolf Cove","Southern Water Tribe"],
	"id":1,
	"eye_color":
	"Cyan",
	"name":"Korra",
	"ethnicity":["Water Tribe"],
	"birthdate":"153 AG"
}

Route

GET /api/characters?perPage=number&page=number

Description

Returns a paginated list of characters by page and pageNumber. No start/end indexing needed.

The endpoint automatically determines the correct starting/ending point for index based off of perPage and pageNumber:

					
if (pageNumber > 1) {
startAt = ((pageNumber - 1) * perPageQuery) + 1;
}

Example Link

/api/characters

Example Response

									
HTTP 200 OK
Content-Type: application/json
[
{
	"name":"Korra",
	"eye_color":"Cyan",

Route

GET /api/characters?nationality=value,value

Description

Returns a list of characters who match the given nationality or nationalities, separated by commas.

Overrides ethnicity filter if given.

Example Link

/api/characters?nationality=Republic City,United Republic

Example Response

									
HTTP 200 OK
Content-Type: application/json
[
{
	"first_episode":"A Leaf in the Wind",
	"id":2,
	"eye_color":"Tangerine",
	"ethnicity": ["Fire Nation","Earth Kingdom"],
	"gender":"Man",
	"image_url":"https://static.wikia.nocookie.net/avatar/images/a/a6/Mako.png/revision/latest?cb=20150408124557",
	"nationality":["Republic City","United Republic"],
	"type":"Human",
	"birthdate":"152 AG",
	"name":"Mako"
}
]
